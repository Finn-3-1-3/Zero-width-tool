01 — Pixel = 3 bytes màu

Mỗi pixel PNG gồm 3 kênh R G B, mỗi kênh 8 bit (0–255). Ảnh 1000×1000 = 1 triệu pixel = 3 triệu bytes màu.

02 — Bit cuối (LSB) không gây khác biệt

Thay đổi bit cuối chỉ làm giá trị màu ±1 (ví dụ 200 → 201). Mắt người không phân biệt được sự chênh lệch 1/255 về độ sáng.

03 — Mỗi byte chứa 1 bit bí mật

Để ẩn 1 ký tự ASCII (8 bit) cần 8 bytes màu ≈ 2.67 pixel. Ảnh 1920×1080 chứa được ~248.000 ký tự.

04 — Header 32-bit lưu độ dài

32 bit đầu tiên lưu số bit của thông điệp. Khi giải mã, đọc header trước → biết chính xác phải đọc bao nhiêu bit → không đọc dữ liệu rác.

05 — Phải lưu định dạng Lossless

Chỉ dùng PNG hoặc BMP. JPEG dùng nén lossy sẽ thay đổi giá trị pixel → mất hết dữ liệu ẩn.
